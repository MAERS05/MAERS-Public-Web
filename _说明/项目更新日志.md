# MAERS - 项目更新日志

## [2026-03-11]

### 💅 界面 (UI) 与 🐛 修复 (Fix)

- **Vditor 脑图渲染修复与全面本地化**：
  - **核心组件本地化**：为解决 CDN 加载缓慢及思维导图（Mindmap/Markmap）无法显示的问题，将 Vditor 及其依赖（ECharts, Markmap, KaTeX, Mermaid 等）资源全面迁移至本地 `plugins/vditor-assets` 目录。
  - **路径结构优化**：建立了 `plugins/vditor-assets/dist` 的规范资源目录，并通过在 `new Vditor` 初始化时强制指定 `cdn: 'plugins/vditor-assets'` 属性，实现了 100% 的内网/离线可用性。
  - **前台渲染升级**：更新了 `custom/cms/admin/cms-editor.module.js`，前台阅读模式 (`_renderVisitorReader`) 废弃了原有的 `marked.js` 方案，改为由 `Vditor.preview` 驱动。现在笔记详情页可以完美呈现思维导图、数学公式、精美代码高亮及深色模式适配。
  - **全站适配**：同步更新了 `notes.html`, `videos.html`, `record.html`, `literature.html`, `games.html` 等页面的资源加载路径，彻底移除外部 CDN 引用。
