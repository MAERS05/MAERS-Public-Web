## [2026-03-17]

### 功能 (Feat) 与 修复 (Fix)

- **Markdown 内部链接跳转支持与重构**：
  - **相关文件**：`custom/cms/admin/cms-editor.module.js`
  - **更新内容**：完全重写引擎拦截逻辑 `_interceptMdLinks(container, currentNode)`。现在阅读器以及后端 Vditor 中点击任意不带 `http` 的本地 `.md` 链接时，能够解析并在当前前端/后端窗口中无缝平滑加载、渲染出该笔记内容（解决网站 404 问题）。
- **跨模块跳转/直达笔记 (`openNote` URL 参数)**：
  - **相关文件**：`main.module.js` (各模块), `admin-main.module.js`
  - **更新内容**：在入口处添加了 URL 参数拦截逻辑。现在可以在 Markdown 文章中书写带参数跳转链接（如 `[黑神话：悟空](/games.html?openNote=黑神话：悟空)`），系统一旦跳转新页面，就会自动读取 `?openNote=` 参数并在渲染结束后自动弹开对应的详情窗口，实现模块间的无感跳转体验。

## [2026-03-11]

### 界面 (UI) 与 修复 (Fix)

- **Vditor 脑图渲染修复与全面本地化**：
  - **相关文件**：`plugins/vditor-assets/dist`, `notes.html`, `videos.html`, `record.html`, `literature.html`, `games.html`
  - **更新内容**：为解决 CDN 加载缓慢及思维导图（Mindmap/Markmap）无法显示的问题，将 Vditor 及其依赖（ECharts, Markmap, KaTeX, Mermaid 等）资源全面迁移至本地 `plugins/vditor-assets` 目录。建立了 `dist` 规范资源目录，通过在 `new Vditor` 初始化时强制指定 `cdn` 属性，实现了 100% 的内网/离线可用性。同步彻底移除外部 CDN 引用。
- **前台渲染升级**：
  - **相关文件**：`custom/cms/admin/cms-editor.module.js`
  - **更新内容**：前台阅读模式 (`_renderVisitorReader`) 废弃了原有的 `marked.js` 方案，改为由 `Vditor.preview` 驱动。现在笔记详情页可以完美呈现思维导图、数学公式、精美代码高亮及深色模式适配。
